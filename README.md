# writing a novel live

*on attention, awareness & the fragmented mind*

---

A novel written in public. One session at a time. Every commit is a session.

This repository is both the novel and the proof-of-work — a visible record of a creative act being built incrementally, in the open, without hiding the process.

The git history is the story behind the story.

---

## the novel

A work of literary fiction exploring attention, awareness, and what it means to think and feel in an age of fragmentation. Written live on YouTube, one focused session at a time.

**Status:** in progress  
**Started:** March 2026  
**Sessions:** ongoing

---

## the project

Each writing session is a commit. Each commit is timestamped proof that the work happened. 

The writing app itself is also built in public. The tool and the work co-evolve.

---

## structure

```
chapters/          — the novel, one .md file per chapter
NOTES.md           — session notes, ideas, fragments
README.md          — this file
```

## run it yourself

If you want to use this writing environment:

```bash
git clone https://github.com/danielclayton-me/writing-a-novel-live.git
cd writing-a-novel-live
npm install
node server.js
```

Then open `http://localhost:3000` in your browser.
